### PAGViewer 快捷键说明

| 快捷键 | 说明 |
| ------ | -------- |
| B | 显示/隐藏 背景色 |
| P | 显示/隐藏 性能检测面板 |
| L | 显示/隐藏 图层编辑面板 |
| Ctrl + P | 导出当前帧为 PNG 文件 |
| 方向键上 ↑ | 暂停并回到首帧 |
| 方向键下 ↓ | 暂停并回到最后一帧 |
| 方向键左 ← | 暂停并回到前一帧 |
| 方向键右 → | 暂停并回到后一帧 |
| 空格键 | 暂停/播放 |
| Commond + W | 关闭当前窗口 |
