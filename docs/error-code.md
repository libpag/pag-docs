---
id: error-code
title: 导出错误码说明
---


导出错误码对照：

| 错误码 | 详细信息 |
| :------: | :------ |
| 0 | 正确。没有错误。|
| -1 | 错误，详情未知 |
| -100 | 导出过程出错 |
| -102 | 编码过程出错 |
| -103 | 解码过程出错 |
| -200 | Tag级别超过了限定的范围 |
| -201 | 没有选择要导出的合成 |
| -202 | 图片数目超过了限定范围(>=50) |
| -203 | webp编码错误 |
| -204 | 导出序列帧时渲染截屏错误 |
| -205 | 导出序列帧过程出错 |
| -206 | 视频编码过程错误 |
| -207 | 序列帧数目太多(>=5)|
